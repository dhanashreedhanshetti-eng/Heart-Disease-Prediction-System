# ❤️ Heart Disease Prediction System

A Machine Learning web application that predicts the likelihood of heart disease using patient health parameters. This project was developed as part of the **CodeAlpha Machine Learning Internship**.

---

## 📌 Project Overview

This application predicts whether a patient is at **High Risk** or **Low Risk** of heart disease using a trained **Random Forest Classifier**.

The web interface is built using **Streamlit**, making it simple and interactive for users.

---

## 🚀 Features

- ❤️ Heart Disease Prediction
- 🤖 Random Forest Machine Learning Model
- 📊 Prediction Confidence Score
- 📈 Progress Bar Visualization
- 🩺 User-Friendly Medical Input Form
- 💡 Health Recommendations
- 📱 Responsive Streamlit Interface
- ⚠️ Medical Disclaimer

---

## 🧠 Machine Learning Model

- **Algorithm:** Random Forest Classifier
- **Accuracy:** 83.60%

---

## 📂 Dataset

**UCI Heart Disease Dataset**

The dataset contains patient information including:

- Age
- Gender
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise-Induced Angina
- Old Peak
- Slope
- Major Vessels
- Thallium Stress Test

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Heart-Disease-Prediction-System.git
```

Move into the project folder

```bash
cd Heart-Disease-Prediction-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
